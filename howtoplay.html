HOW TO PLAY


[1] Solve the enigma code by discovering what letter below corresponds to the real letter.

[2] A tick means you guessed correctly and will replace all corresponding letters to the correct letter.

[3] A cross means you have guessed the incorrect letter for the corresponding letter and to try again.

[4] You can use your mouse or keyboard to input and guess the correct letters.

[5] There are 3 levels. 3 messages need to be completed. Each correct level adds 5 extra seconds onto the timer.

[6] You have 3 minutes to save Britain from nuclear warfare. Goodluck!
